# Super Contact

WoW addon to copy player names from right-click context menus, PvP scoreboards, and auction house seller lists, with bulk whisper and invite support.

## Features

- **Copy Full Name** - Right-click any player → "Copy Full Name" (Name-Realm format)
- **Scoreboard Names** - Button on PvP scoreboard to copy all player names
- **Seller Names** - Button on auction house to copy all seller names
- **Whisper All** - Use `/whisperall <message>` to whisper all players in the current list
