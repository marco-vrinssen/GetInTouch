# CopyAllTheNames

Minimal WoW addon to copy player names from right-click context menus, PvP scoreboards, auction house seller lists, and mouseover text, with bulk whisper and invite support.

## Features

- **Copy Full Name** - Right-click any player → "Copy Full Name" (Name-Realm format)
- **Scoreboard Names** - Button on PvP scoreboard to copy all player names
- **Seller Names** - Button on auction house to copy all seller names
- **Mouseover Copy** - Use `/c` to copy text under the mouse cursor
