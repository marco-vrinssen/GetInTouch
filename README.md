# GetInTouch

Easily copy player names and interact with others in World of Warcraft.

## Features

- **Name Copying**: Copy player names directly from context menus, scoreboards, and AH lists.
- **Context Menu Integration**: Adds 'Copy Name' to right-click menus.
- **Quick Socials**: Streamlined whisper and invite functionality.

## Installation

1. Download the latest version.
2. Extract the files to your `Interface/AddOns` folder.
3. Restart World of Warcraft.
